# Hospital Management System

## 📌 Project Description
This is a web-based hospital management system developed using JSP, Servlet, and JDBC. It helps in managing patient records, doctor details, and appointments.

## 🛠️ Technologies Used
- html,css (Bootstrap)
- Java (JSP, Servlet)
- JDBC
- MySQL
- Apache Tomcat

## ⚙️ Setup Instructions
1. Install Apache Tomcat Server
2. Import project into Eclipse/IDE
3. Configure MySQL database
4. Run the SQL file (hospitaldb.sql)
5. Update database username and password in DBConnection.java
6. Deploy project on Tomcat server

## 🗄️ Database
- MySQL database is used
- Import the provided `hospitaldb.sql` file

## ✨ Features
- Patient Registration
- Doctor Management
- Appointment Booking
- Record Management
- View Prescription
